# Who's Got Mom?
Help -- Grandma fell in her house again! Who's Got Mom is an app to manage who's taking care of your loved one so that you never have another miscommunication about taking care of your favorite family member. 

# Description
'Who's Got Mom' allows you to efficiently schedule, assign, and manage a caretaker for a loved one through a shared calendar app. Users can set up a session event, assign a designated caretaker, and upload information including as dates, time, and general information. It's useful for family and friends alike to keep tabs of those who are a distance away. 'Who's Got Mom' alleviates worry, prevents miscommunication, and most importantly - it keeps you in the loop of your precious loved ones.

# How to Use?
Visit the URL and create an account using your first and last name, email, phone number, additionally you will need to create a password.
You can add squads, members, and create sessions on the calendar for efficient scheduling. From there, it will take you to your dashboard so you can begin creating squads, adding members, and arranging caretakers for your loved ones. 
Additionally, users are able to edit profile and contact information within squads.
